# Docker Compose

refer to the [Docker](./005.docker-aio.md) guide on the specifics of how to work with this image.

## Prerequisites
- [Docker Desktop](https://www.docker.com/get-started) or [Docker Compose](https://docs.docker.com/compose/install/)

## Example docker-compose.yml

### Without ssl

docker-compose.yml
```yaml
version: "3.9"
services:
  nocodb:
    container_name: nocodb
    image: nocodb/nocodb_aio:latest
    environment:
      NC_AUTH_JWT_SECRET: 'your-super-secret-passs'
      NC_PUBLIC_URL: '__your_nocodb_domain__.com'
    ports:
      - "80:8080"
      - "9000:9000"
    volumes:
      - nocodb_data:/var
	tmpfs:
      - /run:nodev,nosuid,exec,mode=0755
volumes:
  nocodb_data:
```

### With ssl

docker-compose.yml
```yaml
version: "3.9"
services:
  nocodb:
    container_name: nocodb
    image: nocodb/nocodb_aio:latest
    environment:
      NC_AUTH_JWT_SECRET: 'your-super-secret-passs'
      aio_ssl_email: 'you@example.com'
      aio_ssl_domain: 'example.com'
      aio_ssl_enable: true
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - nocodb_data:/var
	tmpfs:
      - /run:nodev,nosuid,exec,mode=0755
volumes:
  nocodb_data:
```

## Start nocodb

```bash
docker-compose up -d

```

## Stop nocodb

```bash
docker-compose down

```

## Upgrading from a previous version

```bash
docker compose pull
docker-compose up -d
```
