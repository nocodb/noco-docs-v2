tags: ['Open Source', 'Nix', 'NixOS']

---

if nix is not already installed on your system, we recommend using the
determinate systems nix installer to install nix. if nix is already installed,
you can skip this step.

```bash
curl --proto '=https' --tlsv1.2 -ssf -l https://install.determinate.systems/nix | sh -s -- install
```

once nix is installed, you can run nocodb using the following command:
```bash 
nix run github:nocodb/nocodb/master
```
