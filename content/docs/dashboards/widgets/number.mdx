# Number Widget

## Adding a Number Widget

1. Click **Number** from the widget bar.
2. A metric widget appears on the dashboard.
3. Use the right panel to configure it.

## Data Setup

* **Source Table**: Choose a table (e.g., `Financial Sample`)
* **Records**:

* All Records
* View-specific records
* Filtered records
* **Calculation**:

* Record count
* Field summary (Sum, Avg, Max, Min, etc.)
* **Currency Display**:

* Enable currency symbol (â‚¹, \$, etc.) if the field is numeric

## Display Options

* **Label**: Customize widget label (e.g., *"Total Revenue"*)
* **Decimal Places**: Control number formatting
* **Size**: Small / Medium / Large

> Example: Number widget with `$19,418,950` as total revenue from the dataset.
