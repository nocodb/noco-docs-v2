The dashboard editor provides a flexible canvas where you can arrange widgets. You can drag and drop widgets to position them as needed.

## Widgets

NocoDB dashboards support the following widget types:

| Widget Type     | Description                            |
|-----------------|----------------------------------------|
| **Number**      | Displays KPIs and metric summaries     |
| **Pie Chart**   | Visualizes parts of a whole            |
| **Donut Chart** | Pie chart variant with a hollow center |


### Adding Widgets
Click on Edit Dashboard to enter the dashboard editor. Then, use the widget toolbar to the top to add widgets. You can either click on the widget type or drag and drop it onto the dashboard canvas.
