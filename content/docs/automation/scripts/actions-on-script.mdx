### Rename Script

To duplicate a Script
- Use `...` actions button to the right of the script name in the Scripts list
- Select `Renamne`


### Duplicate Script

To duplicate a Script
- Use `...` actions button to the right of the script name in the Scripts list
- Select `Duplicate`

A copy of the script will be created with a suffix ` - Copy`

### Delete Script

To delete a Script
- Use `...` actions button to the right of the script name in the Scripts list
- Select `Delete`
