### Rename Workflow

To rename a workflow:
- Use `...` actions button next to the workflow name in the Workflows list
- Select `Rename`
- Enter a new name and confirm

---

### Duplicate Workflow

To duplicate a workflow:
- Use `...` actions button next to the workflow name in the Workflows list
- Select `Duplicate`

A copy of the workflow is created with a suffix ` - Copy`.

---

### Delete Workflow

To delete a workflow:
- Use `...` actions button next to the workflow name in the Workflows list
- Select `Delete`
- Confirm deletion

> Deleted workflows cannot be restored.

---

### Enable or Disable Workflow

Control whether a workflow runs automatically.

To enable or disable a workflow:
- Use the toggle switch next to the workflow name
- When disabled, triggers will not execute any actions

---

### View Workflow Execution Logs

Review workflow runs to debug or audit executions.

To view logs:
- Open the workflow
- Navigate to the **Runs** or **Logs** section
- Select a run to view trigger input, node execution status, and outputs
